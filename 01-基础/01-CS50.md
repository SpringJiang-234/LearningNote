#### 编译器

##### 命令行参数

```shell
clang -o hello hello.c
```

要了解具体的参数细节，建议去看文档。

##### 链接库

```shell
clang -o hello hello.c -lsc50
```

C语言原生库和第三方库的区别——只有第三方库要连接命令

##### 编译详解四个步骤

1. 预处理 preprocessing
   - `#`
   - 函数声明被包含在`.h`文件中
2. 编译 compiling
   - C语言翻译为汇编语言
     - 汇编语言
       - 很高效
       - 难以理解且繁琐
3. 汇编 assembling
   - 汇编语言翻译为机器语言
4. 链接 linking
   - 很多`.c`文件的01机器语言汇合到一起


- 字符串其实是以`\0`结尾的字符数组

  - 如何确定字符串的长度

    - 利用`\0`

      ```c
      int n=0;
      while (name[n] != '\0')
      {
      	n++;
      }
      printf("%i\n",n);
      ```

    - `string.h`有方法`strlen`可以得到长度

  - 库`ctype.h`的函数`toupper()`小写转大写

#### 命令行参数，main函数

改变main函数接收参数，可以在命令行输入

在`hello.c`：

```CQL
int main(int argc, string argv[])
{
    printf("hello, %s\n", argv[1]);
}
```

在命令行输入：

```shell
make hello
./hello David 
```

在命令行输出：

```shell
hello, David
```
